Oracle APEX Installation Guide

Date: 20 July 2025

1. Install Java (JDK & JRE)

Remove Existing Java

rpm -aq | grep -i jdk

yum remove <jdk version>
dnf remove <jdk version>

Create Java Directory

mkdir -p /opt/java
chmod -R 777 /opt/java

Upload Java Package

cd /opt/java
chmod -R 777 jdk1.8.0_231.zip
unzip jdk1.8.0_231.zip
chmod -R 777 jdk1.8.0_231

Update Environment Variables

vim /etc/environment

PATH=/usr/bin/:/opt/java/jdk1.8.0_231/bin:/opt/java/jdk1.8.0_231/jre/bin
JAVA_HOME=/opt/java/jdk1.8.0_231
JRE_HOME=/opt/java/jdk1.8.0_231/jre

Reload Environment

source /etc/environment

2. Install Apache Tomcat

Two instances will be used:

  • Tomcat 9091 → Application Server
  • Tomcat 9092 → Jasper Reports (JRE)

Tomcat 9091 Setup

mkdir -p /opt/tomcat9091
chmod -R 777 /opt/tomcat9091/*

cp tomcat9091.service /etc/systemd/system/

systemctl daemon-reload
systemctl start tomcat9091
systemctl enable tomcat9091

firewall-cmd --zone=public --add-port=9091/tcp --permanent
firewall-cmd --reload

Log Directory

mkdir -p /loganalysis/9091/
chmod -R 777 /loganalysis/

Tomcat 9092 Setup

mkdir -p /opt/tomcat9092
chmod -R 777 /opt/tomcat9092/*

cp tomcat9092.service /etc/systemd/system/

systemctl daemon-reload
systemctl start tomcat9092
systemctl enable tomcat9092

firewall-cmd --zone=public --add-port=9092/tcp --permanent
firewall-cmd --reload

3. Create Tablespaces for APEX

CREATE TABLESPACE APEX
DATAFILE 'C:\app\oradata\ORCL\apex1.dbf'
SIZE 1G AUTOEXTEND ON NEXT 100M MAXSIZE UNLIMITED;

CREATE TABLESPACE APEX_FILE
DATAFILE 'C:\app\oradata\ORCL\apex_file1.dbf'
SIZE 1G AUTOEXTEND ON NEXT 100M MAXSIZE UNLIMITED;

4. Install Oracle APEX

Extract APEX

cd /home/oracle
unzip apex_21.2.zip
cd apex

Install APEX

sqlplus / as sysdba

alter session set container = orcl;

@apexins.sql APEX APEX_FILE TEMP /i/
@apxchpwd.sql

Unlock Required Users

ALTER PROFILE DEFAULT LIMIT PASSWORD_LIFE_TIME UNLIMITED;

ALTER USER APEX_PUBLIC_USER IDENTIFIED BY password ACCOUNT UNLOCK;
ALTER USER APEX_LISTENER IDENTIFIED BY password ACCOUNT UNLOCK;
ALTER USER APEX_REST_PUBLIC_USER IDENTIFIED BY password ACCOUNT UNLOCK;

Configure REST Services

@apex_rest_config.sql

5. Install ORDS

Prepare ORDS Directory

cd /home/oracle
mkdir ords
cd ords
unzip ords-21.3.0.266.1100.zip
mkdir config

Rename WAR File

mv ords.war medicare.war

Install ORDS

java -jar medicare.war install

Deploy to Tomcat

cp medicare.war /opt/tomcat9091/webapps/

Copy APEX Images

cd /home/oracle/apex
cp -r images /opt/tomcat9091/webapps/

6. Configure Jasper Reports

Edit Database Configuration

cd /opt/reports_config/jri/conf
vim application.properties

[datasource:default]
type=jdbc
url=jdbc:oracle:thin:@192.168.0.254:1521/msapxpdb
username=medicare_new
password=medicare_new

Configure JRE Path

cd /opt/tomcat9092/webapps/jri/WEB-INF
vim web.xml


oc.jasper.config.home
/opt/reports_config/jri
